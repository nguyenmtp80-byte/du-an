# Student Marketplace

Ứng dụng **mua bán đồ dành cho sinh viên**, xây dựng bằng Flutter.

## Tài liệu dự án

Xem **[FLUTTER_AUTH_GUIDE.md](./FLUTTER_AUTH_GUIDE.md)** để hiểu code Login/Register/Logout (dành cho người mới Flutter).

Xem **[PROJECT_OVERVIEW.md](./PROJECT_OVERVIEW.md)** để biết chi tiết về:

- Cấu trúc Database / API và sơ đồ ERD
- Mô tả từng màn hình (Login, Product List, Cart, Checkout, Sell, Chat, ...)
- Yêu cầu đánh giá từng chức năng
- State Management (Provider / Bloc)
- Sơ đồ điều hướng và cấu trúc thư mục gợi ý

## Getting Started

```bash
flutter pub get
flutter run
```

Tài liệu Flutter: [docs.flutter.dev](https://docs.flutter.dev/)
